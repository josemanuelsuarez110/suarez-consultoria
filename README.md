# SUAREZ | Consultoría & Asesoría Estratégica
> **Liderazgo en Ciberseguridad e Infraestructura Financiera de Élite.**

![Hero Screenshot](https://raw.githubusercontent.com/josemanuelsuarez110/jose-premium-portfolio/main/public/images/jose_suit.jpg)

## 🎯 Visión General
Esta plataforma digital proyecta la identidad profesional de **José Manuel Suárez**, Ingeniero en Sistemas (*Suma Cum Laude*) y Licenciado en Contabilidad. Diseñada bajo una estética **Cyber-Premium**, fusiona la robustez de la ciberdefensa con la precisión de la estrategia financiera corporativa.

## 🚀 Características Principales
- **Bóveda de Credenciales:** Sistema interactivo de verificación de 40+ certificaciones internacionales (Cisco, HP, Python Institute, etc.).
- **Matriz de Servicios Élite:** 
  - Auditoría ISO 27001 & IT.
  - Hacking Ético & Pentesting.
  - Los 5 Pilares de Estrategia Financiera.
- **Insights 2026:** Blog prospectivo sobre IA, Ciberseguridad y Gobernanza corporativa.
- **Diseño Inmersivo:** Glassmorphism, animaciones fluidas con Framer Motion y una identidad visual impactante.

## 🛠️ Stack Tecnológico
- **Frontend:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animaciones:** [Framer Motion](https://www.framer.com/motion/)
- **Iconografía:** [Lucide React](https://lucide.dev/)
- **Despliegue:** [Vercel](https://vercel.com/)

## 📂 Estructura del Proyecto
- `src/components/`: Componentes modulares (Hero, Leadership, CredentialsVault, Blog, etc.).
- `public/diplomas/`: Repositorio estático de certificaciones verificadas.
- `public/images/`: Recursos visuales de identidad de marca.
